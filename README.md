# custom_language_compiler
A small compiler implemented in C using Flex and Bison. Includes lexical analysis, parsing, symbol table generation, intermediate code, and assembly generation for a custom language.

![Example](screenshots/compiler.gif)


## Packages

```bash
$ sudo apt install bison
$ sudo apt install flex
```
